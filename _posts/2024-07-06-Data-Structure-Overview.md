## Data Structure
**자료구조(Data Structure)**는 컴퓨터가 데이터를 효율적으로 다룰 수 있게 도와주는 데이터 보관 방법과 데이터에 관한 연산의 총체이다.

![Data Structure Overview](/assets/img/Data Structure Overview/Data Structure Overview.png)

자료구조에는 단순 자료구조와 복합 자료구조가 있다. **단순(Primitive) 자료구조**는 `int`, `long`과 같은 기본 데이터 형식이다. **복합(Non-Primitive) 자료구조**는 선형 자료구조와 비선형 자료구조가 있다. **선형(Linear) 자료구조**는 데이터 요소를 **순차적**으로 연결하는 자료구조이고, **비선형(Non-Linear) 자료구조**는 데이터 요소를 **비순차적**으로 연결하는 자료구조이다.
